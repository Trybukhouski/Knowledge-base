# Какие существуют типы единиц измерения в CSS

CSS поддерживает два основных типа единиц измерения:

- абсолютные:
  - px
  - cm
  - mm
  - in
  - pt
  - pc
- относительные:
  - %
  - em
  - rem
  - vw
  - vh
  - vmin
  - vmax
  - ex
  - ch
